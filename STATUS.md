# SOFBlck — User Guide (2026-05-21)

## Навигация между экранами

```
[Экономный экран]
      │
      │ BACK (одиночный)
      ▼
[Главный экран] ──── двойной UP   ──▶ [Будильники]
                ──── двойной DOWN ──▶ [Таймеры]
                ──── двойной START──▶ [Секундомеры]
                ──── двойной BACK ──▶ Компас вкл/выкл
                ──── одиночный BACK─▶ [Экономный экран]
```

---

## Экран: Экономный режим

**Файлы:** `EcoView.mc` + `EcoDelegate.mc`

Стартовый экран приложения. Минимальный расход ресурсов.

**Отображает:**
- Ближайший активный таймер: `T ЧЧ:ММ` (оранжевый, выше времени)
- Ближайший активный будильник: `Б ЧЧ:ММ` / `Al ЧЧ:ММ` (жёлтый, выше времени)
- Текущее время `ЧЧ:ММ` крупно по центру экрана
- Дата `дд-мм-ДН` ниже (ДН = день недели: Пн/Вт/…)

**Обновление:** перерисовка раз в минуту; проверка таймеров и будильников каждую секунду.

**Подсветка:** LIGHT включает подсветку на 15 секунд. Повторный LIGHT гасит досрочно.

**Управление:**

| Кнопка | Действие |
|--------|----------|
| BACK | → Главный экран |
| LIGHT | Подсветка вкл (15 сек) / выкл |

---

## Экран: Главный

**Файлы:** `TactixView.mc` + `TactixDelegate.mc`

**Отображает:**
- Фон-глобус (битмап под устройство)
- Аналоговые стрелки (часы, минуты, секунды)
- 7-сегментный дисплей времени ЧЧ:ММ:СС
- Данные по кругу циферблата: батарея (дни), пульс, высота, рассвет/закат, температура, давление, калории, шаги, дата (день недели / число / месяц)
- Пиктограмма + время ближайшего активного будильника (вверху)
- Оверлей секундомера или таймера (когда активны)
- Стрелки компаса N/S (когда активен)

**Обновление:** 1 сек. Подсветка поддерживается принудительно каждую секунду.

**Управление:**

| Кнопка | Одиночное | Двойное (≤500 мс) |
|--------|-----------|-------------------|
| UP | — | → Будильники |
| DOWN | — | → Таймеры |
| START | — | → Секундомеры |
| BACK | → Экономный экран + компас выкл | → Компас вкл/выкл |

---

## Экран: Секундомеры

**Файлы:** `StopwatchListView.mc` + `StopwatchListDelegate.mc`

**Вход:** двойной START на главном экране.

**Отображает:** 5 независимых секундомеров в виде списка.
- Формат времени: `ЧЧ:ММ:СС.сс` (с сотыми секунды)
- Выбранный — жёлтый на тёмно-синем фоне
- Активный (запущен) — `>`, остановленный с временем — `||`

**Обновление:** 40 мс.

**Управление:**

| Кнопка | Действие |
|--------|----------|
| UP | Выбрать предыдущий секундомер |
| DOWN | Выбрать следующий секундомер |
| START (одиночный) | Старт / Стоп выбранного |
| START (двойной) | Сброс выбранного |
| BACK | → Главный экран |

*Состояние сохраняется при закрытии приложения.*

---

## Экран: Таймеры

**Файлы:** `TimerView.mc` + `TimerDelegate.mc`

**Вход:** двойной DOWN на главном экране.

**Отображает:** 5 независимых таймеров обратного отсчёта в виде списка.
- Формат: `ЧЧ:ММ:СС`, статус `>` (идёт) / `||` (пауза) / `!` (истёк)
- Выбранный — жёлтый, истёкший — красный

**Обновление:** 1 сек.

**Режим списка:**

| Кнопка | Действие |
|--------|----------|
| UP | Выбрать предыдущий таймер |
| DOWN | Выбрать следующий таймер |
| START (одиночный) | Старт/Пауза; если не настроен → режим установки |
| START (двойной) | Сброс выбранного таймера |
| BACK | → Главный экран |

**Режим установки времени** (`ЧЧ:ММ:СС`, активное поле жёлтое):

| Кнопка | Действие |
|--------|----------|
| UP | Увеличить значение поля |
| DOWN | Уменьшить значение поля |
| START | Следующее поле; на последнем — запустить таймер |
| BACK | Предыдущее поле; на первом — выйти из режима установки |

*Состояние сохраняется при закрытии приложения.*

## Экран: Уведомление таймера

**Файлы:** `TimerNotificationView.mc` + `TimerNotificationDelegate.mc`

Появляется автоматически при истечении таймера.

**Отображает:** зелёный экран «ТАЙМЕР N / DONE»

**Сигнализация:** вибрация каждую секунду до нажатия START.

| Кнопка | Действие |
|--------|----------|
| START | Сбросить таймер и закрыть |
| BACK | Сбросить таймер и закрыть |

---

## Экран: Будильники

**Файлы:** `AlarmListView.mc` + `AlarmListDelegate.mc` + `AlarmManager.mc`

**Вход:** двойной UP на главном экране.

**Отображает:** 5 слотов будильников.
- Формат: `N. ЧЧ:ММ  ON/OFF [V][S]` (V = вибрация, S = звук)
- Выбранный — жёлтый, включённые — белые, выключенные — серые

**Управление:**

| Кнопка | Действие |
|--------|----------|
| UP | Выбрать предыдущий |
| DOWN | Выбрать следующий |
| START | Открыть подменю |
| BACK | → Главный экран |

**Подменю:** Изменить время / Включён (toggle) / Вибрация (toggle) / Звук (toggle) / Удалить

---

## Экран: Редактор будильника

**Файлы:** `AlarmEditView.mc` + `AlarmEditDelegate.mc`

**Вход:** «Изменить время» в подменю будильника.

**Отображает:** `ЧЧ:ММ` крупно, активное поле жёлтое.

| Кнопка | Действие |
|--------|----------|
| UP | +1 к активному полю |
| DOWN | −1 к активному полю |
| START | Переключить поле (час ↔ мин) |
| BACK | Сохранить и выйти |

---

## Экран: Уведомление будильника

**Файлы:** `AlarmNotificationView.mc` + `AlarmNotificationDelegate.mc`

Появляется автоматически при срабатывании будильника.

**Отображает:** тёмно-красный экран «БУДИЛЬНИК» + время срабатывания.

**Сигнализация:** вибрация + звук (по настройкам) каждую секунду до нажатия START.

| Кнопка | Действие |
|--------|----------|
| START | Остановить сигнал и закрыть |
| BACK | Остановить сигнал и закрыть |

---

## Компас

**Интегрирован в:** `TactixApp.mc` + `TactixView.mc`

**Вход/выход:** двойной BACK на главном экране.  
Также выключается автоматически при переходе в экономный режим (одиночный BACK).

**Отображает:** стрелки N (тёмно-синий) / S (оранжевый) поверх циферблата.  
При недоступности сенсора — сообщение «NO COMPASS» на 2 секунды.

## Хранение данных

| Ключ | Содержимое |
|------|------------|
| `sw5_data` | Массив из 5 секундомеров (offset, running, save_t) |
| `tm5_data` | Массив из 5 таймеров (remain, duration, running, expired, save_t) |
| `v1alarms` | Массив из 5 будильников (hour, min, enabled, vibe, sound) |
