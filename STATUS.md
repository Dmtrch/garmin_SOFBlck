# SOFBlck — User Guide (2026-05-23)

## Реализовано

| Группа | Возможности |
|---|---|
| **Циферблат** | Глобус-фон · аналоговые стрелки · 7-сегментный ЧЧ:ММ:СС · 12 полей данных по кругу · подсветка |
| **Эко-экран** | Минимальный расход · % заряда часов (цвет по уровню) · время · дата · ближайший таймер/будильник |
| **Будильники** | 5 слотов · подменю (время / вкл-выкл / вибрация / звук / удалить) · уведомление с сигналом |
| **Секундомеры** | 5 независимых · точность до сотых · работа в фоне · сохранение состояния |
| **Таймеры** | 5 независимых · режим установки `ЧЧ:ММ:СС` · сохранение состояния · уведомление с вибрацией |
| **Компас** | Стрелки N (синий) / S (оранжевый) поверх циферблата · сообщение `NO COMPASS` при ошибке сенсора |
| **Навигация** | Меню `Компас / Метки / Пеленг` (BACK×2) |
| **Метки (waypoints)** | До 20 точек · три способа добавления (GPS / вручную / нативная карта) · удаление · цвет по индексу |
| **Карта (MapPickView)** | Офлайн OSM с диска часов · pan пальцем/кнопками · три режима LAT/LON/ZOOM · сохранение точки |
| **Пеленг (мульти-цель)** | Выбор нескольких меток · стрелки и дистанции цветом каждой метки · автокоррекция при активном компасе |
| **Справка (UP-hold)** | На всех экранах · автолокализация RU/EN · скролл · 5 пустых строк padding |

---

## Навигация между экранами

```
[Экономный экран]
      │ BACK (одиночный)
      ▼
[Главный экран] ── двойной UP    ─▶ [Будильники] ── START ─▶ [Подменю] ── «Изм. время» ─▶ [Редактор будильника]
                ── двойной DOWN  ─▶ [Таймеры]
                ── двойной START ─▶ [Секундомеры]
                ── одиночный BACK ─▶ [Экономный экран] (+ компас выкл / пеленг стоп)
                ── двойной BACK   ─▶ [Меню навигации]

[Меню навигации] ── «Компас»       ─▶ переключение компаса
                 ── «Метки»         ─▶ [Меню меток] ── текущие GPS / вручную / на карте / удалить
                 ── «Пеленг»        ─▶ [Список меток :pickForBearing]

[Меню меток] ── «Вручную»       ─▶ [Редактор координат]
              ── «На карте»      ─▶ [MapPickView]
              ── «Удалить»       ─▶ [Список меток :pickForDelete]

[Любой экран] ── UP (удержание) ─▶ [Справка]
```

## Экран: Экономный режим

**Файлы:** `EcoView.mc` + `EcoDelegate.mc`

Стартовый экран приложения. Минимальный расход ресурсов.

**Отображает:**
- `% заряда часов` сверху (серый ≥25%, оранжевый ≤25%, красный ≤10%)
- Ближайший активный таймер: `T ЧЧ:ММ` (оранжевый, выше времени)
- Ближайший активный будильник: `Б ЧЧ:ММ` / `Al ЧЧ:ММ` (жёлтый, выше времени)
- Текущее время `ЧЧ:ММ` крупно по центру экрана
- Дата `дд-мм-ДН` ниже (ДН = день недели: Пн/Вт/…)

**Обновление:** перерисовка раз в минуту; проверка таймеров и будильников каждую секунду.

**Подсветка:** LIGHT включает подсветку на 15 секунд. Повторный LIGHT гасит досрочно.

| Кнопка | Действие |
|--------|----------|
| BACK | → Главный экран |
| LIGHT | Подсветка вкл (15 сек) / выкл |
| UP (удержание) | → [Справка] |

## Экран: Главный

**Файлы:** `TactixView.mc` + `TactixDelegate.mc`

**Отображает:**
- Фон-глобус (битмап под устройство)
- Аналоговые стрелки (часы, минуты, секунды)
- 7-сегментный дисплей времени ЧЧ:ММ:СС
- Данные по кругу циферблата: батарея (дни), пульс, высота, рассвет/закат, температура, давление, калории, шаги, дата
- Пиктограмма + время ближайшего активного будильника
- Оверлей секундомера или таймера (когда активны)
- Стрелки компаса N/S (когда компас активен)
- **Стрелки и подписи дистанции до каждой выбранной метки пеленга** — цвет каждой стрелки совпадает с цветом метки в списке; при активном компасе угол корректируется поворотом часов

**Обновление:** 1 сек. Подсветка поддерживается принудительно каждую секунду.

| Кнопка | Одиночное | Двойное (≤500 мс) | Удержание |
|--------|-----------|-------------------|-----------|
| UP | — | → Будильники | → [Справка] |
| DOWN | — | → Таймеры | — |
| START | — | → Секундомеры | — |
| BACK | → Экономный + компас/пеленг стоп | → [Меню навигации] | — |

## Экран: Секундомеры

**Файлы:** `StopwatchListView.mc` + `StopwatchDelegate.mc`

**Вход:** двойной START на главном экране.

**Отображает:** 5 независимых секундомеров в виде списка.
- Формат времени: `ЧЧ:ММ:СС.сс` (с сотыми секунды)
- Выбранный — жёлтый на тёмно-синем фоне
- Активный (запущен) — `>`, остановленный с временем — `||`

**Обновление:** 40 мс.

| Кнопка | Действие |
|--------|----------|
| UP | Выбрать предыдущий секундомер |
| DOWN | Выбрать следующий секундомер |
| START (одиночный) | Старт / Стоп выбранного |
| START (двойной) | Сброс выбранного |
| BACK | → Главный экран |
| UP (удержание) | → [Справка] |

*Состояние сохраняется при закрытии приложения.*

## Экран: Таймеры

**Файлы:** `TimerView.mc` + `TimerDelegate.mc`

**Вход:** двойной DOWN на главном экране.

**Отображает:** 5 независимых таймеров обратного отсчёта в виде списка.
- Формат: `ЧЧ:ММ:СС`, статус `>` (идёт) / `||` (пауза) / `!` (истёк)
- Выбранный — жёлтый, истёкший — красный

**Обновление:** 1 сек.

**Режим списка:**

| Кнопка | Действие |
|--------|----------|
| UP | Выбрать предыдущий таймер |
| DOWN | Выбрать следующий таймер |
| START (одиночный) | Старт/Пауза; если не настроен → режим установки |
| START (двойной) | Сброс выбранного таймера |
| BACK | → Главный экран |
| UP (удержание) | → [Справка] |

**Режим установки времени** (`ЧЧ:ММ:СС`, активное поле жёлтое):

| Кнопка | Действие |
|--------|----------|
| UP | Увеличить значение поля |
| DOWN | Уменьшить значение поля |
| START | Следующее поле; на последнем — запустить таймер |
| BACK | Предыдущее поле; на первом — выйти из режима установки |

*Состояние сохраняется при закрытии приложения.*

| Кнопка | Действие |
|--------|----------|

| DOWN | Выбрать следующий |
| START | Открыть подменю |
| BACK | → Главный экран |
| UP (удержание) | → [Справка] |

**Подменю:** Изменить время / Включён (toggle) / Вибрация (toggle) / Звук (toggle) / Удалить

**Файлы:** `NavMenuDelegate.mc`

**Вход:** двойной BACK на главном экране.

**Пункты (Menu2):**
- **Компас** — переключить включение

## Экран: Меню меток

**Файлы:** `WaypointMenuDelegate.mc`

**Пункты (Menu2):**
- **Текущие координаты** — добавить метку по текущему GPS-фиксу
- **Ввести вручную** → [Редактор координат]
- **Указать на карте** → [MapPickView] (если устройство поддерживает MapView)
- **Удалить** → [Список меток `:pickForDelete`]

Без GPS-фикса пункт «Текущие координаты» сообщает об отсутствии связи со спутниками.

## Экран: Список меток

**Файлы:** `WaypointListDelegate.mc` (содержит `WaypointListView`)

**Режимы:**
- `:pickForDelete` — удаление одной метки
- `:pickForBearing` — мульти-выбор для запуска пеленга

**Отображение:**
- Каждая метка рисуется собственным цветом из палитры `NavManager.colorFor(idx)` (8 цветов циклически)
- Префикс `*` — метка выбрана для пеленга
- Внизу подсказка: `SEL: выбор   HOLD: пуск (N)` — счётчик выбранных
- Справа отображается расстояние от текущего GPS-фикса до каждой метки

| Кнопка | Действие |
|--------|----------|
| UP/DOWN | Прокрутка списка |
| START (короткий) | `:pickForDelete` — удалить; `:pickForBearing` — toggle выбора |
| START (удержание) | `:pickForBearing` — пуск пеленга для всех выбранных; если ничего не выбрано — берётся текущий |
| BACK | → Меню навигации |
| UP (удержание) | → [Справка] |

## Экран: Редактор координат

**Файлы:** `WaypointEditView.mc` (содержит `WaypointEditDelegate`)

**Вход:** «Ввести вручную» в меню меток.

**Отображает:** 6 полей в формате `±DD°.NNNN ±DDD°.NNNN` — знак широты, градусы широты, дробь, знак долготы, градусы долготы, дробь. Активное поле жёлтое.

| Кнопка | Действие |
|--------|----------|
| UP | +1 в активном поле (знак — toggle) |
| DOWN | −1 в активном поле (знак — toggle) |
| START | Следующее поле; после последнего — сохранить и выйти |
| BACK | Выход без сохранения |
| UP (удержание) | → [Справка] |

## Экран: Карта (MapPickView)

**Файлы:** `MapPickView.mc` + `MapPickDelegate.mc`

**Вход:** «Указать на карте» в меню меток (доступно на устройствах с `WatchUi.MapView`).

**Отображает:** нативную офлайн-карту OSM с диска часов с крестом-курсором в центре. Bbox ±500 м вокруг текущей позиции. Текстовый оверлей с координатами и текущим режимом.

| Кнопка | Действие |
|--------|----------|
| Touch drag | Pan карты |
| UP | Шаг 50 м по активной оси (вверх / +) |
| DOWN | Шаг 50 м по активной оси (вниз / −) |
| START | Переключить режим (LAT → LON → ZOOM) |
| SELECT | Сохранить точку как метку |
| BACK | Выход без сохранения |

Справка по UP-hold на этом экране **не привязана** — `onMenu()` занят переключением оси.

## Экран: Справка (HelpView)

**Файлы:** `HelpView.mc` + `HelpContent.mc`

**Вход:** удержание кнопки UP на любом экране (кроме MapPickView).

**Отображает:**
- Заголовок экрана (жёлтый)
- Список пар «КЛАВИША: действие» на языке системы (RU/EN)
- Индикаторы прокрутки `^` / `v` если контент не помещается
- Подсказка внизу: `^v скролл  <- выход`

**Особенности отрисовки для круглого экрана:**
- 5 пустых строк padding сверху и снизу — для скролла строк в безопасную центральную зону
- Боковые отступы `≈ ширина 3 символов` шрифта `FONT_XTINY` — текст не уходит за круглый край

| Кнопка | Действие |
|--------|----------|
| UP | Скролл вверх |
| DOWN | Скролл вниз |
| BACK | Выход |

## Компас

**Интегрирован в:** `TactixApp.mc` + `TactixView.mc`

**Вход/выход:** через [Меню навигации] → «Компас», либо автоматически выключается при переходе на экономный экран.

**Отображает:** стрелки N (тёмно-синий) / S (оранжевый) поверх циферблата.
При недоступности сенсора — сообщение «NO COMPASS» на 2 секунды.

При включённом компасе стрелки активного пеленга автоматически корректируются с поворотом часов.

---

## Пеленг (мульти-цель)

**Интегрирован в:** `TactixApp.mc` (state) + `TactixView.mc` (отрисовка) + `WaypointListDelegate.mc` (выбор) + `NavManager.mc` (расчёты)

**Состояние:** массивы `bearingTargetIndices`, `bearingDistancesM`, `bearingDirectionsRad`.
`startBearing(indices)` принимает массив индексов меток.
`onPositionUpdate` в цикле обновляет все цели.

**Отрисовка:** для каждой цели — стрелка и подпись дистанции (`Nm` или `K.Nkm`) цветом waypoint'а из `NavManager.colorFor(idx)`.

**Остановка:** одиночный BACK на главном экране.

## Хранение данных

| Ключ | Содержимое |
|------|------------|
| `sw5_data` | Массив из 5 секундомеров (offset, running, save_t) |
| `tm5_data` | Массив из 5 таймеров (remain, duration, running, expired, save_t) |
| `v1alarms` | Массив из 5 будильников (hour, min, enabled, vibe, sound) |
| `nav_wp` | Массив до 20 меток (`{name, lat, lon}`) |

## Палитра цветов меток

`NavManager.colorFor(idx)` — циклически по индексу:

| idx | Цвет | Hex |
|-----|------|-----|
| 0 | green | `0x00FF00` |
| 1 | cyan | `0x00FFFF` |
| 2 | magenta | `0xFF00FF` |
| 3 | orange | `0xFFAA00` |
| 4 | yellow | `0xFFFF00` |
| 5 | pink | `0xFF0055` |
| 6 | lime | `0xAAFF00` |
| 7 | light blue | `0x55AAFF` |
