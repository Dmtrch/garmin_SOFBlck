TACTIX BLACK — Tactical-Style Analog Watch Face

Globe background, analog hands, 7-segment digital display. 5 stopwatches, 5 timers, 5 alarms, compass, multi-target bearing to 20 waypoints. Context-sensitive help on every screen (UP hold).

━━ ECO SCREEN (startup) ━━

Low-power screen. Battery % top, HH:MM center, date, nearest timer and alarm. GPS and sensors suspended.

  LIGHT → backlight 15 s · BACK → main screen · UP hold → help

━━ MAIN SCREEN ━━

Full face: globe, analog hands, HH:MM:SS, 12 data fields, compass, bearing arrows. GPS indicator in dial center when active — color shows signal quality: green (good), olive (usable), orange (poor), yellow (last known), red (unavailable).

  START → GPS on/off · START×2 → stopwatches
  UP×2 → alarms · DOWN×2 → timers
  BACK → eco · BACK×2 → navigation menu · UP hold → help

━━ 5 STOPWATCHES ━━

Independent, hundredths precision, run in background.

  UP/DOWN → select 1–5 · START → start/pause · START×2 → reset · BACK → exit · UP hold → help

━━ 5 TIMERS ━━

Independent countdown timers. Expiry: full-screen alert + vibration.

  UP/DOWN → select / ±1 in field · START → start/pause / next field
  START×2 → reset · BACK → prev field/exit · UP hold → help
  Alert: START or BACK → stop and reset

━━ 5 ALARMS ━━

Per-alarm time, on/off, vibration, sound. Next alarm shown on main dial.

  UP/DOWN → select · START → alarm menu · BACK → exit · UP hold → help
  Edit time: UP/DOWN → ±1 · START → next field · BACK → save and exit
  Alert: START or BACK → stop

━━ NAVIGATION MENU (BACK×2) ━━

Compass — live N/S arrows over dial, toggle on/off.

Waypoints — up to 20 named points. Add by:
  • Current GPS position
  • Manual lat/lon entry (UP/DOWN digit, START next, BACK prev, DOWN hold → toggle DD°MM′SS″/DD.DDDD)
  • Offline OSM map (drag to pan, UP/DOWN step/zoom, START → cycle lat/lon/zoom mode, START×2 → save)
  Confirmation screen previews coords. Then name up to 8 chars.
  Delete: pick from list with confirmation.

Waypoint list: UP/DOWN scroll · START×2 delete · BACK exit · UP hold help

Bearing — select waypoints (START toggle, BACK activate). Live direction arrows and distances on dial. Auto-corrects with compass. Eco screen suspends GPS but keeps bearing — main screen restores instantly.
