TACTIX BLACK — аналоговый циферблат в тактическом стиле

Строгий циферблат с фоном-глобусом, аналоговыми стрелками и 7-сегментным дисплеем. Экономный экран, 5 будильников, 5 секундомеров, 5 таймеров, компас.

━━ ЭКОНОМНЫЙ ЭКРАН (стартовый) ━━

Минимальное отображение: время ЧЧ:ММ крупно по центру, дата дд-мм-ДН. Выше времени — ближайший активный таймер (T ЧЧ:ММ) и будильник (Б ЧЧ:ММ). Сигналы будильников и таймеров срабатывают здесь. LIGHT — подсветка на 15 с.

  BACK        → главный экран

━━ ГЛАВНЫЙ ЭКРАН ━━

Глобус · аналоговые стрелки · 7-сегментный ЧЧ:ММ:СС · 12 полей данных (батарея, пульс, высота, давление, температура, рассвет/закат, шаги, калории, дата) · будильник · секундомер/таймер · компас.

  BACK (одиночный)  → экономный экран + компас выкл
  BACK × 2          → компас вкл/выкл
  START × 2         → 5 секундомеров
  UP × 2            → 5 будильников
  DOWN × 2          → 5 таймеров

━━ 5 СЕКУНДОМЕРОВ (START×2) ━━

Точность до сотых секунды (ЧЧ:ММ:СС.сс). Продолжают отсчёт в фоне.

  UP / DOWN       — выбор секундомера
  START           — старт / стоп
  START × 2       — сброс
  BACK            — выход

━━ 5 ТАЙМЕРОВ (DOWN×2) ━━

Пять независимых обратных отсчётов. При срабатывании — зелёный экран, вибрация до отключения.

  UP / DOWN       — выбор таймера / изменение значения
  START           — старт/пауза; если не настроен → установка времени
  START × 2       — сброс
  BACK            — выход / предыдущее поле установки

Уведомление таймера: START — выключить сигнал.

━━ 5 БУДИЛЬНИКОВ (UP×2) ━━

До 5 будильников с настройкой вибрации и звука. Ближайший активный — на циферблате.

  UP / DOWN       — навигация по списку
  START           — меню (время / вкл-выкл / вибрация / звук / удалить)
  BACK            — выход

Уведомление будильника: START — выключить сигнал.

━━ КОМПАС (BACK×2) ━━

Стрелки N (синий) / S (оранжевый) поверх циферблата. BACK×2 или BACK (одиночный с главного) — выкл.

TACTIX BLACK — Tactical-style analog watch face

Globe background · analog hands · 7-segment display. Eco screen, 5 alarms, 5 stopwatches, 5 timers, compass.

━━ ECO SCREEN (startup) ━━

Minimal view: large HH:MM centered, date dd-mm-DY. Nearest active timer (T HH:MM) and alarm (Al HH:MM) shown above. Alarms and timers fire here. LIGHT — backlight 15 s.

  BACK        → main screen

━━ MAIN SCREEN ━━

Globe · analog hands · 7-segment HH:MM:SS · 12 data fields (battery, HR, elevation, pressure, temp, sunrise/sunset, steps, calories, date) · alarm · stopwatch/timer overlay · compass.

  BACK (single)   → eco screen + compass off
  BACK × 2        → compass on/off
  START × 2       → 5 stopwatches
  UP × 2          → 5 alarms
  DOWN × 2        → 5 timers

━━ 5 STOPWATCHES (START×2) ━━

Hundredths precision (HH:MM:SS.ss). Keep running in background.

  UP / DOWN   — select · START — start/stop · START×2 — reset · BACK — exit

━━ 5 TIMERS (DOWN×2) ━━

Five independent countdowns. On expiry: green alert screen, vibration until dismissed.

  UP/DOWN — select/adjust · START — start/pause or enter setup · START×2 — reset · BACK — exit
  Notification: START — stop alert.

━━ 5 ALARMS (UP×2) ━━

Up to 5 alarms with vibration and sound options. Nearest shown on dial.

  UP/DOWN — navigate · START — menu (time/on-off/vibration/sound/delete) · BACK — exit
  Notification: START — dismiss.

━━ COMPASS (BACK×2) ━━

N (blue) / S (orange) arrows on dial. BACK×2 or single BACK from main — turns off.
